The intent of the classes in this folder is to implement simplified integration of the Rollbar Notifier within ASP.NET (Full Framework - not Core) MVC and WebApi.

Implement helper method for capturing and reporting a HttpResponseException;
Implement Rollbar's System.Web.Mvc.IExceptionFilter
Implement Rollbar's System.Web.Http.Filters.IExceptionFilter
Implement Rollbar's System.Web.IHttpModule
